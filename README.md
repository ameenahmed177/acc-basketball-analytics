# ACC Basketball Analytics Project

## Overview
This project develops a statistical model to predict point spreads for ACC men's basketball games using historical NCAA data.

## Tools Used
- R (data processing, modeling, prediction)
- hoopR (data extraction)

## Data
- NCAA men's basketball data (2022–2025 seasons)
- Includes both conference and non-conference games involving ACC teams

## Model
- Linear regression model using home and away team effects
- Captures relative team strength and home-court advantage

## Predictions
- Generated point spread predictions for 78 ACC games
- Applied name-mapping to align competition templates with model data

## Prediction Intervals
- Constructed prediction intervals using model residual variance
- Targeted ~70% coverage based on historical calibration

## Goal
Provide interpretable predictions of game outcomes and quantify uncertainty using statistical modeling.
